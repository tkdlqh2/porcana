Porcana Admin Wireframe ADMIN

Reference screens for admin operations and support flows.

Core
1. Dashboard
Operational overview.
2. User List
Search and review users.
3. User Detail
Inspect user details and ownership.
4. Asset List
Asset operations and edits.
5. Portfolio List
Portfolio search and filtering.
6. Portfolio Detail
Holdings and performance detail.
7. Batch List
Batch execution history.
8. Batch Detail
Inspect batch issues and logs.
9. Inquiry List
Track incoming inquiries by status and user.
10. Inquiry Detail
Review thread, update status, and reply.

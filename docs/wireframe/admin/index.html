Porcana Admin Wireframe ADMIN

관리자 대시보드 화면 프로토타입

대시보드
1. 대시보드
전체 현황 요약 - 회원 수, 종목 수, 포트폴리오 수 통계
회원 관리
2. 회원 목록
회원 검색, 역할 필터링, 페이지네이션
3. 회원 상세
회원 정보, 포트폴리오 목록, 아레나 이력
종목 관리
4. 종목 목록
종목 검색, 시장 필터, 이미지/배당 정보 수정
포트폴리오 관리
5. 포트폴리오 목록
전체 포트폴리오 조회, 이름 검색
6. 포트폴리오 상세
포트폴리오 정보, 보유 종목, 수익률
Admin API Endpoints (참고)
POST /api/v1/admin/admins - 관리자 생성
GET /api/v1/admin/users - 회원 목록
GET /api/v1/admin/users/{userId} - 회원 상세
DELETE /api/v1/admin/users/{userId} - 회원 탈퇴
GET /api/v1/admin/assets - 종목 목록
PATCH /api/v1/admin/assets/{assetId}/image - 이미지 수정
PATCH /api/v1/admin/assets/{assetId}/dividend - 배당 수정
GET /api/v1/admin/portfolios - 포트폴리오 목록
GET /api/v1/admin/portfolios/{portfolioId} - 포트폴리오 상세

# Porcana

**Portfolio + Arena**: 투기장처럼 종목을 선택하여 쉽게 포트폴리오를 만들고 수익률을 추적하는 서비스

## 개요

Porcana는 포트폴리오 단위로 투자에 접근하는 새로운 방식을 제안합니다. Hearthstone의 Arena 모드에서 영감을 받아, 사용자가 재미있고 직관적인 방식으로 자신만의 투자 포트폴리오를 구성할 수 있도록 돕습니다.

## 핵심 기능

- **Arena 방식 종목 선택**: 매 라운드마다 3개의 종목 카드 중 하나를 선택하여 포트폴리오(Deck)를 구성
- **포트폴리오 추적**: 구성한 포트폴리오의 수익률을 실시간으로 모니터링
- **메인 Deck**: 가장 관심 있는 포트폴리오를 메인으로 설정하여 홈 화면에서 바로 확인
- **성과 차트**: 기간별(1M/3M/1Y) 포트폴리오 및 개별 종목 성과 시각화
- **간편한 포트폴리오 관리**: DRAFT, ACTIVE, FINISHED 상태로 포트폴리오 생애주기 관리
